# SmartEats-landing
SmartEats es una landing page orientada a estudiantes universitarios y jóvenes profesionales que buscan reducir la fatiga causada por malos hábitos alimenticios.
El objetivo del proyecto es ofrecer información clara y accesible sobre cómo mejorar la alimentación diaria mediante guías y consejos prácticos, promoviendo un mayor bienestar, energía y rendimiento académico/laboral.
